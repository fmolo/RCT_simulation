# Code repository for Scholl Molo, No cause for concern
The purpose of randomization is not to create balance.

- example_anova.pdf documents the generation of Figure 1. It is created by example_anova.qmd.

- RCT_simulation.pdf documents the generation of Figure 2. It is created by RCT_simulation.qmd

- get_clinicaltrials_gov_data.R calculates the median number of participants in clinical trials registered on clinicaltrials.gov, informing the data simulation in Section 5.3.

The figures can alternatively be generated by running the .R scripts instead of the .qmd files.
